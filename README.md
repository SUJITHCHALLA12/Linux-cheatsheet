#  Linux Basics for Hackers – Interactive Command Reference

An interactive, chapter-wise command reference inspired by the book **Linux Basics for Hackers**.
This project transforms traditional learning into a **clickable, searchable, and structured web experience** for faster revision and deeper understanding.

---

##  Live Demo

👉 Add your deployed link here
`https://sujithchalla12.github.io/linux-cheatsheet/`

---

## 📌 Features

*  **Chapter-wise organization** (based on book structure)
*  **Real-time search & filtering**
*  **Clickable commands → detailed explanations**
*  Includes:

  * Syntax
  * Purpose
  * Common usages
  * Important flags
  * Real-world examples
*  **Instant UI updates (no reloads)**
*  Clean, responsive interface

* All commands are stored in a structured **JavaScript data file**
* UI dynamically:

  * Filters commands
  * Renders lists
  * Displays detailed panels
* No backend required — fully **client-side**
